# ShapeTutor

A browser-based shape-matching game built as a personal challenge to build a web game. Themed after KPop Demon Hunters for my niece Zi, a
2-year-old who is very much obsessed with the movie.

## How to Play

Match 3 of the same shape in a row. Select 3 matching shapes to clear them. Select 3 non-matching shapes to deselect and try again.

## Stack

Vanilla HTML, CSS, and JavaScript — no frameworks or build tools.

## Structure

- `index.html` — start screen
- `game.html` — game screen
- `style.css` — shared styles (KPop Demon Hunters dark theme)
- `assets/` — shape images for characters
